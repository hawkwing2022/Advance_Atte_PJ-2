## 機能一覧

-   ユーザー登録機能
-   ログイン機能
-   ログアウト機能
-   ユーザー名表示機能
-   4 つの打刻機能(勤務開始時刻、勤務終了時刻、休憩開始時刻、休憩終了時刻)
-   日付毎の勤怠情報一覧表示機能
